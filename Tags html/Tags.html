Títulos e Subtítulos:

As tags de título funcionam de forma hierárquica crescente, onde a tag "h1" é mais importante que a tag "h2" e etc. Cada tag h tbm tem um tamanho
diferente entre elas. Visualmente as tags de título não tem diferenca, porém há valor semântico.


Parágrafos:

Os parágrafos são representados pela tag "p" e vão até o final da página. Para fazer uma quebra de linha manual é utilizada a tag "br". Essa TAG não possui fechamento.

Para deixar uma palavra ou frase em negrito é utilizada a tag "b".


Listas:

Não ordenadas

As listas NÃO ordenadas são feitas com a tag "ul", para adicionar uma lista Ñ ordenada e a tag "li" para representar os Itens da lista.
EX.:

  • Item 1
  • Item 2
  • Item 3

Ordenadas

Parte do mesmo princípio das listas não ordenadas, porém a tag que forma a lista é a "ol". A lista ordenada vem com um type padrão como "1", mas pode receber diversos atributos no type.
EX.:

  1. Item 1
  2. Item 2
  3. Item 3
  1. Item 1
  2. Item 2
  3. Item 3
  1. Item 1
  2. Item 2
  3. Item 3

Listas de descrição

Esse tipo de lista é declarado com a tag "dl". Sua diferença dos outros é que essa lista recebe um ITEM e uma Descrição do item, utilizando as tags "dt" e "dd", respectivamente.
EX.:

Item 1:
Descrição do item 1.
Item 2:
Descrição do item 2.
Item 3:
Descrição do item 3.

Lembrando que também é possível colcoar listas dentro de outras listas.

Tags meta:

As tags meta possuem diversos atributos. O mais conhecido é o "charset". Outro atributos são:

name="description"
Utilizado para adicionar uma descrição a sua página.
name="keywords"
Adicionar palavras chaves à sua página.
name="author"
Adicionar o criador da página.
http-equiv="refresh"
utilizado para sua página atualizar sozinha

PS: tudo é escrito no atributo content="" após escrever o atributo desejado acima.

Legendas

Para adicionar legendas no seu vídeo é preciso seguir os passos a seguir:

  1. Criar tag "track".
  2. Adicionar os atributos:
    • kind="subtitles"
    • srclang="pt-br"
    • label="PT-BR"
    • src="nomeArquivo.vtt"
  3. Fechar tag "track".
  4. Criar um arquivo com o nome nomeArquivo.vtt .

Neste novo arquivo você irá escrever:

WEBVTT

Olá, tudo bem?(essa será a legenda)
